# GrainLab

Film Photography Effect Simulator — 胶片摄影效果模拟器

## Features

- 🎞️ Film grain simulation
- 🌈 Color grading
- 💡 Light leak effects
- 🔲 Vignette
- 🎨 Fade effects
- 📦 Batch processing
- 🖼️ Before/After comparison
- 🌍 i18n (English / 中文)

## Tech Stack

- Vue 3 + TypeScript
- Vite
- Pinia
- Vue I18n

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
